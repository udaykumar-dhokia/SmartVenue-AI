# SmartVenue AI

AI-powered venue intelligence platform for the Narendra Modi Stadium, Ahmedabad. Real-time crowd management, smart routing, and personalized attendee experiences during IPL cricket matches.

## Phase 2 Features Included

- **Interactive Seat Navigation**: Attendees can enter their exact seat (e.g. "J-3", "401", "PAV L-2") to see it highlighted on the map with gate and walk-time info.
- **Admin Simulation Controls**: Venue staff can play/pause the crowd simulation, adjust speed (0.5x - 4x), override specific zone densities, and trigger global events like Halftime Surge or Evacuation.
- **Live Attendee Reviews**: Attendees can submit 1-5 star reviews that appear instantly in the Admin Dashboard with sentiment analysis.
- **Dual Attendee Layout**: Toggle between a mobile phone preview and a full-width responsive web interface.

## Architecture

The application is a Next.js (App Router) web application with two primary views:

### Admin Command Center

Full-screen operations dashboard for venue staff:

- **Simulation Controls**: Top control bar to manage the venue state manually or automatically.
- **Interactive Stadium Map**: Real `map.svg` of the Narendra Modi Stadium. Click zones or adjust simulation to see density heatmaps update.
- **AI Agent Feed**: Live stream of autonomous agent decisions generated by Google GenAI (Gemini 2.5 Flash).
- **Stats Panel**: KPI cards showing attendance, average wait time, crowd safety score, flow efficiency.
- **Alerts Panel**: Active alerts with severity-based color coding.
- **Concession Panel**: Real-time outlet status showing wait times and stock levels.
- **Live Reviews**: Incoming feed of attendee feedback with overall sentiment breakdown.

### Attendee Companion

Available in Mobile Frame or Full Web layout:

- **Stadium Map & Seat Finder**: Search "K-5" to get routing info and see the section pulse on the map.
- **Smart Nudges**: AI-generated contextual suggestions.
- **Food Pre-Order**: Full menu with cart functionality and nearest outlet recommendation.
- **Digital Ticket**: GT vs RCB match ticket card.
- **AI Chat Assistant**: Natural language chat interface powered by Gemini.
- **Review Submission**: 5-star rating form to send feedback to admins.

| Layer | Technology |
|---|---|

| Language | TypeScript |
| Styling | Tailwind CSS v4 + shadcn/ui |
| Animations | Framer Motion |
| AI | Google GenAI SDK (`@google/genai`) with Gemini 2.5 Flash |

## Environment Variables

Create a `.env.local` file in the `app/` directory:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

Get your API key from [Google AI Studio](https://aistudio.google.com/).

## Getting Started

```bash
cd app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
